# Contributing

Thank you for considering contributing to Standardoc CLI!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Install dependencies:
```bash
bun install
```

4. Build the project:
```bash
bun run build
```

5. Run tests:
```bash
bun test
```

## Development

- Use `bun run watch` for development with auto-rebuild
- Use `bun run dev` to test the CLI locally

## Code Style

- Follow TypeScript strict mode
- Use meaningful variable names
- Add JSDoc comments for public APIs
- Keep functions focused and small

## Submitting Changes

1. Create a branch for your changes
2. Make your changes
3. Test thoroughly
4. Submit a pull request with a clear description

## Questions?

Open an issue for questions or discussions.
